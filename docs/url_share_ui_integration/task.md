# タスクリスト

## 目的

- 共有導線をわかりやすいUIで追加する。
- 共有URLをクリップボードコピー/他アプリ共有できるようにする。
- `#s=<payload>` 付きURLアクセス時にデータを自動ロードする。

## 実施項目

- [x] データ管理セクションに「共有リンクを作成」ボタンを追加。
- [x] 共有シート（コピー/他アプリ共有/JSON保存）を追加。
- [x] 共有URLの長さ表示・警告表示を追加。
- [x] 共有URL生成失敗時のエラーハンドリングを追加。
- [x] クリップボードコピー失敗時のエラーハンドリングを追加。
- [x] `#s=<payload>` 自動ロード処理を `App` に追加。
- [x] 自動ロード成功後にURLハッシュを消去する処理を追加。
