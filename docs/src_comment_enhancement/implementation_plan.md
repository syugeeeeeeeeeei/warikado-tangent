# 実装計画

1. `src/` 配下の TypeScript/TSX ファイルを走査し、コメント追加対象を特定する。
2. ルート（`main.tsx` / `App.tsx`）と型定義（`types/domain.ts`）に責務説明コメントを追加する。
3. ユーティリティ（`utils`）とフック（`hooks`）に、入出力・境界条件・分岐理由を説明するコメントを追加する。
4. 画面コンポーネント（`views`）に、イベントハンドラと画面遷移意図のコメントを追加する。
5. 表示コンポーネント（`components`）に、表示条件・操作導線・UI上の優先ロジックを説明するコメントを追加する。
6. `yarn build` でビルド確認を行い、コメント追加のみであることを差分で確認する。
