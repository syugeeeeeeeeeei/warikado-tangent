# タスクリスト

## 依頼内容

- イベント名を20文字、メンバー名を10文字に制限する。
- 制限時のメッセージハンドリングを追加する。
- JSON圧縮/展開ロジックを作成する。
- 複数ケースのJSONテストデータを作成し、圧縮/展開テストを行う。
- テストで得た文字列長データを分析する。
